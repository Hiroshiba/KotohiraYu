# Getting started

1. Python3の仮想環境を作成します

   ```virtualenv -p python3 venv```

2. 必要パッケージをインストールします

   ```pip3 install -r requirements.txt```

3. config.iniを設定する

4. accesstoken.txtにアクセストークンを入れる

5. KotohiraYu.pyを実行する

これだけですっ！
